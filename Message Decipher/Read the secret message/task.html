Description

Welcome to the first stage, Agent! Your mission, should you choose to accept it, is to gather encrypted messages from a confidential text file: secret-message.txt and log them to the console.

Using the power of Node.js, you will navigate through the digital realm, accessing the hidden secrets within the file. Your task is to extract the encrypted messages, which are encoded to keep them secure.

With precision and skill, you will command the console to display the encrypted messages. Each line of code you execute brings you closer to unraveling the mystery. The console becomes your ally, showcasing the encrypted secrets as you decode the digital puzzle.

Your objective is to ensure that the program successfully reads the encrypted messages from the file and displays them accurately in the console. This stage sets the foundation for the subsequent decryption operations that lie ahead.

After logging the contents of the file, output this message: Finished reading the file to let other agents know that the process is finished!

Good luck, Agent. This message will self-destruct in 5... 4... 3... Just kidding! Let's get started!

Description

Welcome back, Agent, to the second stage of the project! Your mission intensifies as you have found out that the message is using the Atbash cipher for encryption.

The Atbash cipher is a special technique that swaps each letter of the alphabet with its counterpart from the opposite end. For example, 'A' becomes 'Z', 'B' becomes 'Y', and so on. This method helps us transform the encrypted messages into their original form.

Let me provide you with an example of how the Atbash cipher works. Suppose we have the following encrypted message: 

Svokg rh z tlrmt ozgvh.

To decode this message using the Atbash cipher, we replace each letter with its counterpart from the opposite end of the alphabet. Applying the cipher, we get:

Hello is a glint light.

Now, with this powerful knowledge, you will employ Node.js to apply the Atbash cipher algorithm to every character of the encrypted messages. As a result, the console will display the decrypted messages, revealing their hidden meaning.

After decoding the message log this new message instead of the old one: Finished decoding the file

Watch out for the cases of the letters, make sure to match them with the original text!

Each decrypted message brings you closer to the truth, shedding light on the covert operations and hidden agendas. This stage marks a significant breakthrough in our mission, as we gain access to the decrypted messages, ready to be analyzed and deciphered further.

Good luck, Agent. Let the decryption operation commence!
